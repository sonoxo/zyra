AI PENTESTING AGENT

Create an AI-driven pentesting agent capable of performing safe and authorized vulnerability discovery.

The agent must:

analyze application structure
identify potential attack surfaces
generate safe test payloads
execute automated vulnerability testing
analyze responses
document findings

Pentesting tests must include:

SQL injection
cross-site scripting (XSS)
CSRF vulnerabilities
authentication weaknesses
API security issues
misconfigured access controls

The AI pentest agent must produce detailed attack diagnostics explaining:

what vulnerability was tested
how the vulnerability was detected
what payload was used
what response was received
recommended mitigation steps

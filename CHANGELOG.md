# Changelog

All notable changes to Zyra will be documented in this file.

## [1.0.0] - 2026-04-01

### Added
- Monorepo structure with apps/web and apps/api
- Prisma schema for users, profiles, streams, payments
- JWT authentication (register, login, me)
- Profiles CRUD API
- Streams CRUD API with go-live/end-stream
- Chat messaging system
- Stripe payments integration
- AI Agents package (Pentest, Threat Detection, Incident Response)
- Full CI/CD workflow
- Docker + docker-compose ready

### Changed
- Rebranded from Sentinel Forge to Zyra
- Updated README with "How Zyra Works" section

### Security
- Added SECURITY.md for vulnerability reporting
- Enabled Dependabot for dependencies
